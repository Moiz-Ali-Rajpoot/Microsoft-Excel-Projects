# 🧑‍💼 Employee Dashboard — Microsoft Excel

> **Individual Employee Profile & Attendance Intelligence** — A fully interactive, modern dashboard built entirely in **Microsoft Excel**, demonstrating advanced Excel skills including dynamic lookups, conditional formatting, and clean UI design without any BI tool.

---

## 🗂️ Overview

This is a **single-employee profile dashboard** built in Microsoft Excel that displays a complete 360° view of an individual employee — covering personal details, attendance summary, official information, salary, and reporting hierarchy — all from one clean, interactive screen.

The dashboard features a **search/filter bar** at the top to switch between employees dynamically, making it fully reusable across an entire organization. Key highlights include:

- **Attendance Summary** — Present (96% | 77 days), Leaves (3% | 2), Late Arrivals, Half Days, and Others tracked with circular progress indicators
- **Personal & Official Profile** — Employee ID, Role, Department, Position Level, Location, Joining Date, Tenure, and contact details displayed in a structured layout
- **Salary & Allowances Panel** — Current Salary and Travelling Allowance with an upcoming increment notification alert
- **Attendance Breakdown Bar Chart** — Full breakdown across all leave types: Casual, Earned, Medical, Unpaid, Maternity, Paternity, Bereavement, Umrah, Exam, Earned Leave, Suspended, Labour Day, and more
- **Reporting Hierarchy Panel** — Visual org chart showing the employee's chain of command from Chief Operating Officer → Director Operations → Assistant Director Ops → Manager Operations → Team Lead

## 💡 Excel Expertise Demonstrated

This project showcases **advanced Excel dashboard development** skills:

- 🎨 **Modern UI Design in Excel** — Clean layout with color themes, icons, rounded card-style panels, and a macOS-inspired aesthetic — all built natively in Excel without any third-party tools
- 🔍 **Dynamic Employee Search** — A search bar at the top filters and loads any employee's complete profile instantly using lookup formulas
- 📊 **Interactive Charts** — Attendance breakdown chart updates dynamically based on the selected employee
- 🔔 **Smart Alerts** — Conditional notifications such as "Upcoming Increment" trigger automatically based on data logic
- 🏢 **Hierarchy Visualization** — Reporting chain rendered visually inside Excel using shapes and linked data
- 📋 **Multi-Category Attendance Tracking** — Covers 15+ leave and attendance types with formula-driven calculations
