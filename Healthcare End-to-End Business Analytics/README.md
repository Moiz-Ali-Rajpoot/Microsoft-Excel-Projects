# 🏥 Healthcare End-to-End Business Analytics Dashboard — Microsoft Excel

> **Complete Practice Management Intelligence** — A fully automated, multi-source Excel dashboard system built with **Advanced Formulas, Power Query, and M Query transformations** — delivering a 360° business overview across KPIs, Revenue Cycle, Charges & Payments, and Appointment Operations for data-driven healthcare practice management.

---

## 🗂️ Overview

This is the most comprehensive dashboard in this portfolio — a **three-in-one Healthcare Practice Management Dashboard** that consolidates data from **EHR (Electronic Health Record) software** and **Google Sheets** into a single automated Excel source via Power Query. It gives practice owners, billing managers, and operations leads a complete real-time view of their business — from referrals and revenue down to individual appointment categories and insurance distributions.

Filterable by **Year and Month**, all three dashboards refresh simultaneously, making it a true end-to-end business intelligence solution built entirely in Microsoft Excel.

## ⚙️ Advanced Technical Features

- 🔗 **Multi-Source Data Integration** — EHR software exports and multiple Google Sheets are pulled, merged, and unified into a single Excel data model via Power Query — completely eliminating manual data consolidation
- 🧹 **Advanced M Query Transformations** — Custom M Query scripts handle complex data cleaning: standardizing EHR export formats, resolving Google Sheets inconsistencies, merging patient records across systems, and reshaping raw data into structured analytical tables
- 🔄 **Fully Automated Pipeline** — One refresh button pulls the latest data from all sources, cleans and transforms it, and updates every KPI, chart, table, and trend across all three dashboards instantly
- 📐 **Dynamic MOM% & YOY% Calculations** — Every KPI card automatically calculates Month-over-Month and Year-over-Year percentage changes with directional indicators
- 🎨 **Color-Coded Conditional Formatting** — No Shows highlighted in orange, Cancellations in red, positive trends in green — applied automatically based on data thresholds without manual intervention
- 🔢 **PC Ratio Tracking** — Payment-to-Charge ratio calculated and tracked month by month as a core revenue cycle health indicator
- 📊 **Dual-Axis Trend Analysis** — Charges vs Payments and Referrals vs Scheduled shown on synchronized dual-line charts that update with every month selection
