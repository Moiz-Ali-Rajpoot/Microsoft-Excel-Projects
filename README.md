## 🗂️ Overview

This repository is a collection of **production-level Excel dashboards** built to solve real business problems. Each dashboard demonstrates that Microsoft Excel — when used at an advanced level — can deliver the same intelligence and visual quality as dedicated BI tools like Power BI or Tableau, without requiring additional software licenses or infrastructure.

Every dashboard in this repo follows the same pipeline architecture:

**Raw Data Sources → Power Query ETL → Cleaned Single Source → Advanced Formula Layer → Modern Interactive Dashboard**

## ⚙️ Technical Approach

### 🔗 Multi-Source Data Integration
Raw data is pulled from multiple disconnected sources — EHR software exports, live Google Sheets, survey form responses, and CSV files — and unified into a **single consolidated Excel data model** via Power Query. No manual copy-pasting. No formula errors from stale data.

### 🧹 Advanced Power Query & M Query Transformations
Every pipeline includes custom M Query scripts that handle real-world data messiness: inconsistent date formats, duplicate records, mismatched naming conventions, multi-sheet merges, conditional column logic, and reshaping wide data into analytical structures.

### 🔄 Automated Data Pipelines
Once built, each dashboard is fully automated. Refreshing the data connection pulls the latest information from all sources, cleans it, and updates every KPI card, chart, table, and trend line across the entire dashboard — in seconds.

### 🧮 Advanced Formula Architecture
Complex multi-condition aggregations, dynamic period comparisons (MOM%, YOY%), rolling calculations, and drill-down logic are all built using Excel's native formula engine — no VBA or macros required.

## 🎨 Design Philosophy

Every dashboard in this repo is built with a **modern, user-friendly UI** — designed to be used directly in business meetings without any modifications:

- **Consistent dark/professional color themes** across all panels
- **KPI cards** with current value, MOM%, This Year, and Last Year comparisons
- **Color-coded conditional formatting** — highlights exceptions, risks, and top performers automatically
- **Interactive filters** — Month, Year, Team, Location, and Status selectors refresh all panels simultaneously
- **Navigation tabs** mimicking BI tool page switching for a seamless experience
- **Clean typography and visual hierarchy** — dense data made readable at a glance
